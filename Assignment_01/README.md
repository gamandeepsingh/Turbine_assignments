# Assignment 01

Mint an SPL token and an MPL Core NFT on Solana devnet.

## Setup

```bash
npm install
```

Make sure you have the Solana CLI installed and set to devnet.

## Run

```bash
npm run mint-token
npm run mint-nft
```

## Notes

- The wallet is stored in `wallet.json`.
- Both scripts auto-fund the wallet if needed.
- NFT metadata currently uses a placeholder URI.
